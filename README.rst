Описание
-----------
Менеджер для работы с чатом

Установка пакетом
-----------
Для локальной разработки::
    pip install -e packages/as_chat

Импорт
-----------
Проверка::
    from as_chat.models import Chat


Удаление
-----------
Удалить пакет::
    pip uninstall as_chat

Для создания пакета
https://docs.python.org/3.10/distutils/introduction.html#distutils-simple-example
https://docs.python.org/3.10/distutils/sourcedist.html
::
    python setup.py sdist
